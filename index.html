Runtime Overview

Expected Processes

These should be launched by your Space entrypoint or Docker startup.

`server.js`

WebSocket server for leader/follower sync. Default port is 8080 unless you override it.

`headless.js`

Headless bot controller. Runs as a CLI process inside the container.

Space Health

What This Page Does

  • Auto-fills a WebSocket URL based on the current host.
  • Shows connection state and raw message activity.
  • Helps verify your Space is keeping `server.js` alive.

Tip: If the status stays offline, confirm your Space exposes the same port as the WebSocket server.

Quick Start (Space Runtime)

  • Ensure Node 18+ is available in the Space image.
  • Launch `server.js` and `headless.js` from your Space entrypoint.
  • Open this page and connect to the WebSocket URL shown above.
